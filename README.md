# pc2t

Databáza študentov a učiteľov - Projekt na BPC-PC2T

# Features

- Pridávanie osôb (učiteľ / študent)
- Priradenie učiteľ - študent / študent - učiteľ
- Zadávanie známok
- Vyhľadávanie osôb
- Zoradený výpis
- Výpočet platov učiteľov
- Výpis nákladov školy
- SQL databáza (Read, Create, Delete, Find)
